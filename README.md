# PPL Workout Recommendation

## 概要

Pythonで作成した筋トレメニュー推薦プログラムです。

ユーザーが Push・Pull・Legs からトレーニング部位を選択すると、その日のトレーニングメニューや種目の種類、目的に応じた推奨レップ数を表示します。

## 作成目的

自身の筋トレ経験を活かし、トレーニングメニュー選択をサポートするプログラムを作成しました。

また、Pythonの条件分岐やユーザー入力処理の学習を目的として作成しました。

## 使用技術

* Python
* input
* if / elif / else
* f-string

## 機能

* トレーニング部位選択（Push / Pull / Legs）
* トレーニングメニュー表示
* 多関節種目・単関節種目の判定
* トレーニング目的選択
* 推奨レップ数表示
* 入力内容に応じた結果表示

## 工夫した点

ユーザーの入力内容によって表示結果が変わる条件分岐を実装しました。

また、Push・Pull・Legsそれぞれで異なるトレーニングメニューを表示できるようにしました。

さらに、筋肥大と筋力向上で推奨レップ数を変更し、実際のトレーニングで活用できる内容を意識しました。

## 実行方法

python workout.py
