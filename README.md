PPL Workout Recommendation

概要

Push（胸・肩・三頭筋）、Pull（背中・二頭筋）、Legs（脚）からトレーニング部位を選択し、おすすめの種目や回数を表示するPythonプログラムです。

開発目的

Pythonの学習の一環として、条件分岐やユーザー入力の処理を実践的に学ぶために作成しました。

使用技術

* Python
* input()
* if / elif / else
* ネストした条件分岐
* f-string

主な機能

* Push / Pull / Legs の選択
* 種目の選択
* 筋肥大または筋力向上の選択
* 目的に応じた推奨回数の表示
* 不正な入力への対応

実行方法

1. Pythonをインストール
2. プログラムファイルをダウンロード
3. ターミナルで実行

python ppl.py

実行例

Push・Pull・Legs を選択してください: Push
種目を選択してください:
ベンチプレス
目的を選択してください:
筋肥大
推奨回数: 8～12回
