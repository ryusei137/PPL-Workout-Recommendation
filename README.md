# PPL Workout Recommendation

## 概要
Pythonで作成した筋トレメニュー推薦プログラムです。

ユーザーがpush・pull・legsからトレーニング部位を選択すると、
その日のトレーニングメニューや種目の種類、多関節種目・単関節種目、
目的に応じた推奨レップ数を表示します。

## 作成目的

自身の筋トレ経験を活かし、
トレーニングメニュー選択をサポートするプログラムを作成しました。

## 使用技術

- Python
- input
- if / elif / else
- f-string

## 工夫した点

ユーザーの入力内容によって表示結果を変える条件分岐を実装しました。

また、筋肥大と筋力向上で推奨レップ数を変更できるようにしました。

## 実行方法

python workout.py
